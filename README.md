# WEB DESIGN PROJECT

## DELUXE HOTEL

This project is a responsive web design for Deluxe Hotel, a hypothetical luxary hotel with various sections and pages including the homepage, blogs page and contact us page. The website is designed to be user friendly and visually appealing on different devices such as desktop, tablet and mobile phones.
